# Mew
Just a small test project to test Python packaging  
Install with  
`pip install -e .`  
or  
`python setup.py install`  
while you are inside the directory of this project

Run with  
`mew` or `python -m mew`

Examples:
`mew`  
> *Mews softly*  
`mew you`  
> *Mews softly at you*  
`mew you 'your pets'`  
> *Mews softly at you and your pets*  
`mew fox birb doge`  
> *Mews softly at fops, birb and shork*

*Mews*
